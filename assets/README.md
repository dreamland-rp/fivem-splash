# Dreamland RP Assets

Place your custom assets in this directory:

## Required Files

### logo.png
- Your server logo
- Recommended size: 300x300px or higher (square format)
- Format: PNG with transparent background preferred
- Used in: Navigation and hero section

### hero-video.mp4 (Optional)
- Background video for the hero section
- Recommended: 1920x1080 or higher
- Format: MP4 (H.264 codec)
- Keep file size reasonable for web (under 10MB if possible)
- If not provided, a gradient background will be used as fallback

## Optional Assets

You can add additional images, videos, or other media files here as needed for future expansion.

## Usage

These assets are referenced in the main HTML file and will automatically be loaded when present.
